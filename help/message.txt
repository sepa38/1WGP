```
!set_home_channel
```
全体用のコマンドを実行するためのチャンネルを設定するコマンドです。


```
!new_game
```
新しくゲームを始めることができます。
このコマンドにより BOT から送信されるメッセージにリアクションをつけることで参加登録が完了します。
このリアクションを消すと参加登録を取り消すことができます。
初めてリアクションをつけた時、テキスト、画像送受信用の個人チャンネルが作成されます。


```
!confirm
```
コマンド送信時に参加登録をしていたメンバーでゲームを開始します。


```
!send_subject
```
お題、絵の説明を送信するコマンドです。


```
!send_picture
```
絵を送信するコマンドです。


各コマンドの使用例は添付した画像の通りです。
